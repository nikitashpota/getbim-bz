# 2.3 Инструкция по работе с перемычками

## Настройка типа перемычки

### Формирование наименования типа

Имя типа создается по шаблону, компоненты соединяются через `_`:\
`[Марка]_[ШиринаПроема]_[ОсновнойУголок]_[Полоса]_[ОпорныйУголок]`

**Пример:** `Пр-1_Шп-1000_Уг 100х8_Пл 80х5_Опор уг 100х8лв`

**Ключевые правила:**

* **Марка:** `Пр-1`
* **Ширина проема:** `Шп-1000` (число в мм)
* **Основной уголок:** `Уг 100х8` (см. п.2)
* **Полоса:** `Пл 80х5` (аналогично уголку)
* **Опорный уголок:** `Опор уг 100х8лв` (добавьте `лв`/`пв`/`общ`)

### Правила для уголков и полос (параметры «Тип основного уголка», «Тип полосы»)

**Формат строгий:** `[Размер]x[Толщина]`

* Только числа и **русская «х»**.
* Без пробелов, текста, единиц измерения.

**Примеры:** `80x6`, `100x8`\
**Ошибки:** `80x6` (латинская x), `80 х 6`, `Уголок 80x6`, `121x5` (нет в ГОСТ).

### Параметры геометрии

* **Ширина проема** — основная длина между опорами.
* **Длина опирания 1 (справа)** и **Длина опирания 2 (слева)** — глубина опирания на стены.
* **Важно:** `Общая длина = Ширина проема + Длина опирания 1 + Длина опирания 2`.

Заполняйте значения в миллиметрах согласно расчету/чертежу.
