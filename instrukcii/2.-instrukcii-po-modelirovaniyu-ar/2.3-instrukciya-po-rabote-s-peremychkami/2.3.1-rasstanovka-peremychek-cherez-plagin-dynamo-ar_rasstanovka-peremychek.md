# 2.3.1 Расстановка перемычек через плагин Dynamo "АР\_Расстановка перемычек"

### Краткое описание скрипта

Ссылка на скрипт: [https://disk.yandex.ru/d/AqvhEALJD9DdCA](https://disk.yandex.ru/d/AqvhEALJD9DdCA)

**Что делает:** Автоматически расставляет стальные перемычки над дверями в кирпичных стенах на заданном уровне.

**Фильтрация дверей (что игнорируем):**

1. Двери не на выбранном уровне
2. Двери в стенах без кирпича (проверяем материалы в структуре стены)
3. Двери, над которыми уже есть перемычка (проверка по координатам ±100 мм)
4. Дубликаты дверей

**Логика выбора перемычки:**

* **Толщина стены ≤150 мм** → "ОБЩ\_Перемычка\_Сталь\_1 Уголок"
* **Толщина стены >150 мм** → "ОБЩ\_Перемычка\_Сталь\_2 Уголка"

**Размер уголка зависит от толщины стены:**

* **≤80 мм** → Уг 80х6
* **≤120 мм** → Уг 100х8
* **>120 мм** → Уг 100х8

**Создание типов перемычек:**

* Автоматически создает типы с именами: `Шп-1000_Уг 80х6` или `Шп-1100_2хУг 100х8_Пл 80х5`
* Ширина округляется до 50 мм (шаг)
* Если тип уже существует - использует его

**Параметры размещения:**

* Высота: `Высота нижнего бруса` + `Высота` двери → записывается в "Высота проема" перемычки
* Ширина: берется из параметра двери → записывается в "Ширина проема" типа перемычки
* "Отметка от уровня" = 0 (всегда)
* Ориентация копируется с двери
