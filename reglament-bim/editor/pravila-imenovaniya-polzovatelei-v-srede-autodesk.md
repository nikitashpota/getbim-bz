# Правила именования пользователей в среде Autodesk

В среде Autodesk рекомендуется использовать следующую схему для именования пользователей:

```
ФАМИЛИЯ_ИМЯ_ОТЧЕСТВО_РОЛЬ
```

**Пример:**

```
Иванов_Петр_Александрович_АР
```
